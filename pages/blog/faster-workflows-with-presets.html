Faster Workflows with Presets

Presets repeat kaam ko repeatable bana dete hain. Jitna kam decision‑making, utna fast output. Yeh guide preset governance, examples, aur team sharing tips cover karta hai.

What to preset

  • Dimensions + formats
  • Quality ranges
  • Naming tokens
  • Output folders

Governance

  • Owner per preset (updates accountable)
  • Version tags: hero@v2, thumb@v1
  • Changelog short notes

JSON example

{
  "name": "blog-image",
  "width": 1600,
  "format": "avif",
  "quality": 50,
  "naming": "{project}_{seq2}_{w}x{h}_{fmt}"
}
