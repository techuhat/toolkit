Handle Transparent Backgrounds Correctly

Transparency powerful hai — lekin correctly handle karna zaroori hai taaki halos, matte issues, ya color fringing na aaye.

Basics

  • PNG alpha for UI icons
  • WebP transparency for web graphics
  • Matte only when compositing against known background

Premultiplied vs straight alpha

Wrong alpha handling se edges pe halos aa sakte hain. Export me straight alpha prefer karein; compositing pipeline me premultiply sahi stage pe ho.

Edge cleanup

  • Fringe removal tools use karein
  • 1–2 px contract feather for soft selections
  • Against dark/light test background preview

Formats and use‑cases

  • Logos: SVG/PNG
  • Drop‑shadow assets: PNG/WebP lossless
  • Photos with alpha: WebP/AVIF (preview carefully)

Wrap‑up

Transparency right = clean visuals. Edge checks + correct formats se problems khatam.
