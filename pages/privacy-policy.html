1. Introduction

Welcome to ImageToolkit Pro! Humara mission hai aapko powerful image aur PDF processing tools provide karna jo completely privacy-focused hain. This Privacy Policy explains humari data practices aur aapke rights.

ImageToolkit Pro ek client-side application hai, matlab sab processing aapke browser ke andar hoti hai. Hum aapke files ko kabhi hamare servers par upload nahi karte.

2. Information We Do NOT Collect

Your uploaded files or processed images/PDFs

Personal identification information (name, email, phone)

File metadata or content analysis

Location data or device identifiers

Account creation or user profiles

3. How Our Service Works

Client-Side Processing

All image compression, format conversion, PDF operations, aur other processing tasks aapke browser ke andar hote hain. Modern browsers ke Web APIs (Canvas, Web Workers, File API) ka use karke hum sophisticated processing provide karte hain.

No Server Upload

Your files never travel to our servers. Processing results directly aapke device par generate hote hain aur browser ke download mechanism se save hote hain.

4. Technical Information We May Collect

While hum aapke files collect nahi karte, basic website analytics ke liye minimal technical information collect kar sakte hain:

Usage Analytics: General tool usage patterns (which tools are used most)

Performance Metrics: Loading times, error rates (anonymous)

Browser Information: Browser type and version for compatibility

Important: This data is aggregated, anonymous, aur aapki files ya personal content se bilkul unrelated hai.

5. Cookies and Local Storage

Essential Cookies

Basic website functionality ke liye minimal cookies use karte hain (theme preferences, language settings).

Browser Storage

Your processing preferences (compression levels, output formats) locally save ho sakte hain for better user experience. Ye data aapke device par rehta hai.

6. Third-Party Services

Content Delivery Network (CDN)

Fast loading ke liye hum CDN services use kar sakte hain. Ye services sirf static assets (CSS, JavaScript) serve karti hain, aapke files access nahi karti.

Analytics

Website improvement ke liye privacy-focused analytics tools use kar sakte hain. Personal data ya file content kabhi track nahi hota.

7. Your Rights and Controls

Complete Control: Your files completely aapke control mein rehti hain

No Account Required: No registration, no personal data

Clear Browser Data: Browser settings se preferences clear kar sakte hain

Opt-out Anytime: Stop using the service anytime without any data concerns

8. Security Measures

HTTPS Encryption

All communication encrypted hai. Lekin since processing local hai, network transmission minimal hai.

No Data Retention

Since hum files store nahi karte, data retention ka issue hi nahi hai. Browser close karne par sab clear ho jata hai.

Secure Processing

Modern browser security features ka full advantage lete hain for safe processing environment.

9. Data Retention

Simple answer: Hum koi data retain nahi karte.

Your files temporary browser memory mein process hote hain aur immediately after processing browser se clear ho jate hain. Koi permanent storage nahi hai.

10. Children's Privacy

Humari service general audience ke liye hai. Since hum personal data collect nahi karte, children's privacy ke liye additional concerns nahi hain. Lekin 13 years se kam age ke children ko parental guidance recommended hai.

11. International Users

ImageToolkit Pro worldwide available hai. Since processing local hai, data sovereignty issues nahi hain. Your data aapke device se bahar nahi jata, so international transfer ka question hi nahi uthta.
